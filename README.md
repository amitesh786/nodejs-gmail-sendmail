# Gmail Webapp
Web application using Vue.js, Node.js and MySQL2.

Website: ''

+ How to run on localhost:3000
	+ cd `frontend`
	+ npm run `build`
	+ copy `dist` folder to `backend`
	+ cd `backend`
	+ npm run `develop`

+ How to run on frontend & backend separately:
	+ cd `frontend`
	+ npm run `develop`
	+ localhost: 8080
	+ cd `backend`
	+ npm run `develop`
	+ localhost: 3000
